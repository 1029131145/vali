#vali.js
中文API地址：http://1029131145.github.io/vali/

#使用方法
```javascript
<!-- 非常简单 -->
<form class="form">
    <input type="text" email="t" vali>
</form>
<script type="text/javascript">
    $(".form").vali();
</script>
```
